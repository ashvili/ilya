# neurolithology - ml

## installation

1. Download project

2. Go to project root
   ```shell
   cd /path/to/project/root 
   ```

3. Create python venv
   ```shell
   python3 -m venv venv
   ```
   
4. Initialize python venv
   ```shell
   source venv/bin/activate
   ```

5. Install dependencies
   ```shell
   pip install --upgrade pip
   pip install -r requirements.txt
   ```
